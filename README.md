## Instrucciones de uso
Comando para inicializar el container:
    
Abrir la terminal en la carpeta de la aplicacion web
```bash
$ docker build -t="enlazadin" .
$ docker-compose up -d
```

Comando parar parar el container:
```bash
$ docker-compose stop
```
